# 🏫 Школьная информационная система (School Management System)

Профессиональный инструмент, разработанный для автоматизации работы завуча школы. Проект объединяет в себе логику распределения учебного времени и визуализацию школьных данных.

### 🌟 Основной функционал:
- **Интерактивное расписание:** Динамическое отображение уроков для всех классов и учителей.
- **База данных сотрудников:** Полный список преподавательского состава с фильтрацией.
- **Статистика и аналитика:** Визуализация данных о наполняемости классов, нагрузке учителей и общих показателях школы.
- **Удобный интерфейс:** Быстрый доступ к редактированию и поиску нужной информации.

### 🛠 Технологический стек:
- **Frontend:** HTML5, CSS3 (Адаптивная верстка под планшеты и ПК завуча).
- **Logic:** JavaScript (Обработка массивов данных, фильтрация и поиск).

### 📈 Почему это важно:
Этот проект заменяет бумажные журналы и сложные таблицы, сводя все данные в единый удобный интерфейс, что экономит до 5 часов рабочего времени завуча в неделю.
